Findable Help

A

New

For single page Web applications or any set of Web pages, if one or more of the following help mechanisms is supported, then access to at least one option is included in the same relative order on each page:

  • Human contact details;
  • Human contact mechanism;
  • Help content provided by the author;
  • A fully automated contact mechanism.

Access to help mechanisms may be provided directly on the page, or may be provided via a direct link to a different page containing the information"
