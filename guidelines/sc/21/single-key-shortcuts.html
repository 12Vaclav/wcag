Character Key Shortcuts

A

Proposed

This is a proposed Success Criterion that has not yet been formally approved by the Working Group. Discussion of the issue is available in Issue 69 and Pull Request 133. To file comments on this proposal, please raise new issues for each discrete comment in GitHub.

If a shortcut consisting entirely of character keys is implemented by the web page, then a mechanism is available to turn it off or to remap it to a shortcut that uses at least one non-printing key.
