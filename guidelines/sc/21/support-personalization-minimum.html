Support Personalization (minimum)

A

Proposed

This is a proposed Success Criterion that has not yet been formally approved by the Working Group. Discussion of the issue is available in Issue 6 and Pull Request 125. To file comments on this proposal, please raise new issues for each discrete comment in GitHub.

A version of the content is available such that one of the following is true:

  • It serves the same purpose as the content; important information comes before other information; there are a maximum of 5 controls per screen; and controls have icons and visible text labels.
  • Contextual Information that can be conveyed through author settable properties or meta-data is available for essential functionality and content. Critical features can be programmatic determined.
