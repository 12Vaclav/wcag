Purpose of controls

A

Proposed

This is a proposed Success Criterion that has not yet been formally approved by the Working Group. Discussion of the issue is available in Issue 6. To file comments on this proposal, please raise new issues for each discrete comment in GitHub.

In content implemented using markup languages, the conventional name of conventional user interface components can be programmatically determined. (AA)
