Target Size

AA

Proposed

This is a proposed Success Criterion that has not yet been formally approved by the Working Group. Discussion of the issue is available in Issue 60. To file comments on this proposal, please raise new issues for each discrete comment in GitHub.

The size of the target in relation to the visible display at the default viewport size is at least 44px by 44px for pointer inputs except:

  • an alternative link or control has 44px by 44px touch target
  • link is part of a block of text and it has 44px x 22px touch target

where px is a CSS pixel when the page is using the device ideal viewport.

Editor's Note: this criterion borrows the distinction of "coarse" and "fine" pointing devices from Media Queries Level 4 - Pointing Device Quality: the pointer feature [[!mediaqueries-4]].
