Add text to a control stating all content is editable.

Metadata

3.2.6 Hidden Controls

Sufficient

Applicability

Any technology

Description

The objective of this technique is to provide the information needed to identify a control when all contents of a larger component are editable. This primarily applies to a large design canvas with multiple editable components with it.

To meet this technique, add text that:

  • is programatically and visually adjacent to the component, and
  • states that all content is editable.

    • As a best practice, also add a short instruction on how to edit the components as well though this is not required to meet the Success Criteria.

Examples

Design Canvas

A design canvas allows a user to edit all the design elements with in it. A user must hover over an element to access the controls. If all the controls are displayed, it will clutter the page. Instead, text instructions are included at the top of the canvas stating that all elements within it can be edited by using the menu available on hover or focus.

Tests

Procedure

For controls that progress or complete a process:

  1. Identify a component where all sub-components within it are editable
  2. Confirm that all sub-components include controls that only display when pointer hover or mouse focus is over them.
  3. Confirm that the component includes text that explains that all content within it is editable
  4. Confirm that the text is visually and programmatically adjacent to the component that contains the editable components.

Note: Where a page has multiple page variations (i.e., in a responsive design) controls should be tested in each variation.

Expected Results

  • #3 and #4 are true.
